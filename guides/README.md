# Guides

This directory contains the repository's curated guides. Each document in this folder has been checked against the current project layout so it can be used as public-facing documentation on GitHub.

## Available Guides

- [`project-overview.md`](./project-overview.md): Product scope, repository intent, and the main review paths.
- [`data-assets.md`](./data-assets.md): What data is bundled, what is not, and where generated outputs live.
- [`webapp-guide.md`](./webapp-guide.md): Local setup and refresh workflow for the browser-based application.
- [`streamlit-demo-guide.md`](./streamlit-demo-guide.md): Running the decision-support dashboard against bundled CSV exports.
- [`digital-twin-guide.md`](./digital-twin-guide.md): Running the twin scripts and Streamlit interfaces.
- [`validation-status.md`](./validation-status.md): Status of the legacy `00_README` documents and how each one was handled.
